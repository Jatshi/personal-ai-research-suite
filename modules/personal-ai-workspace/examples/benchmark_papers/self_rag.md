# Self-RAG: Learning to Retrieve, Generate, and Critique through Reflection

Source: https://arxiv.org/abs/2310.11511

Self-RAG trains a language model to retrieve on demand, generate, and critique
its own output with reflection tokens. The model can decide when retrieval is
useful instead of retrieving for every prompt. Reflection tokens assess
relevance, support, and utility. This adaptive behavior aims to improve factual
generation while avoiding unnecessary retrieval cost.
