# AI Retrieval Benchmark Paper Cards

This directory contains concise, reproducible fact cards for public AI papers.
They are not paper PDFs or copied full text. Each card links to its canonical
paper and records the structured facts used by the benchmark questions.

Use this small corpus to verify the benchmark pipeline offline. For a portfolio
claim, rerun `run_benchmark.py --papers <your-reviewed-paper-directory>` against
your own 10-20 legally obtained papers and retain the generated JSON report.
