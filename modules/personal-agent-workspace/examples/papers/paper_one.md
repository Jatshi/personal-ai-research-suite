# Local Agent Safety for Academic Workflows

Authors: Jian Demo, Mei Researcher

Year: 2026

Keywords: agent safety, dry-run, human approval, academic workflow

## Abstract

This paper studies a local AI agent system for academic workflows with safe file operations.

## Introduction

The task is to support doctoral finishing, file organization, and paper reading.

## Method

The method uses a unified tool registry, shared state, path guard, dry-run plans, human approval, audit logs, and rollback records.

## Experiment

The experiment uses simulated messy files, thesis samples, and mock papers. Metrics include safety coverage, task completion, and traceability.

## Discussion

The limitation is that mock LLM summaries are heuristic and should be replaced by a stronger local or API model for production.

## Conclusion

Local agent safety can make personal academic automation more reliable.
