# 简历项目介绍

**ScholarMind Agent：面向个人科研工作流的本地 RAG + AI Agent + MCP 智能工作台**

设计并实现 ScholarMind Agent，一个面向博士论文收尾、学术文献管理、个人知识库问答和本地 AI Agent 工作流的综合型智能工作台。项目采用 Python、Streamlit、FastAPI、SQLite、Chroma、BM25、OpenAI-compatible LLM/Embedding API 和 MCP-style Tool Calling 架构，整合了本地学术 RAG、个人 Agent 工作流、Personal AI Workspace 和 MCP 工具网关四个子系统，支持多格式文档解析、混合检索、重排序、引用溯源、证据校验、安全工具调用和外部 AI 客户端接入。

项目支持 PDF、Word、PPT、Markdown、TXT 等多格式材料导入，可对博士论文、论文笔记、会议记录、简历、项目材料和文献资料进行本地建库。系统实现了 BM25 关键词检索、向量语义检索、Hybrid Search、规则 Reranker、confidence score 和 citation grounding，能够在问答时返回可追溯的文件名、页码/段落和 chunk_id，并在证据不足时拒答，降低 RAG 幻觉风险。

在 Agent 层，项目实现了本地文件整理 Agent、博士论文收尾检查 Agent、多 Agent 论文阅读工作流和个人工作助理 Agent。系统通过统一 Tool Registry 管理工具能力，支持 dry-run、安全路径限制、Human-in-the-loop 审批、文件操作 audit log 和 rollback 记录，能够完成文件扫描、重复文件识别、重命名建议、论文结构检查、图表/参考文献编号检查、论文阅读笔记生成、日报/周报生成和任务规划等工作流。

项目进一步实现了 MCP-style 本地工具网关，将知识库检索、问答、文件系统操作和代码仓库分析能力暴露为结构化工具，便于接入 Claude Desktop、IDE Agent 或其他 MCP-compatible 客户端。系统提供 Streamlit UI、CLI、FastAPI 服务、Docker/CI 配置、生产级 config、OpenAI-compatible API 接入、Mock 离线模式、测试集和中英文使用/开发文档，形成可本地运行、可扩展、可公开发布到 GitHub 的完整 AI Agent 工程项目。
